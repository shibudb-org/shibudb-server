# ShibuDB Changelog

## Version History

### 1.0.1 - 2025-08-17
- Improved vector engine performance
- Added delete operation in vector storage
- Added synchronous wal in vector for durability

### 1.0.0 - 2025-08-13
- Initial release
- Basic database functionality
- FAISS vector engine support
- Key-Value engine support
- Support for multiple platforms (Linux AMD64/ARM64, macOS ARM64)
